Lesson 2 — ES Modules & Imports

What are ES Modules?

ES Modules let you split your code into smaller, reusable files. Use export to expose values and import to consume them.

Named vs default exports

  • Named export: export function add() {} — imported with import { add } from './math.js'
  • Default export: export default class Widget {} — imported with import Widget from './widget.js'

Quick examples

// math.js
export function add(a, b) { return a + b; }

// main.js
import { add } from './math.js';
console.log(add(2, 3));

Why modules help

Modules improve organization, make testing easier, and help tools like bundlers and tree-shakers remove unused code.
